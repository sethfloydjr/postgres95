# postgres95 Cookbook

Cookbook for installing postgresql-95 on RHEL or Debian OSs. At the time this was written the public OS repos did not have this version.

## Requirements

### Platforms

- RHEL or Debian

### Chef

- Chef 11.0 or later

### Cookbooks

- `default`
